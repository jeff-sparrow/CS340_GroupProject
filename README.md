# UltraMarathon Aid Station Management System

An application for managing aid stations in large races consisting of many aid stations, volunteers, and supplies.

## Description

This application was built for CS340: Database Programming. It uses MySQL, React, Express as the main technology stack, and Node as the package manager. This application implements a user interface for managing a database for marathon style races that have aid stations, volunteers, and supplies. The ERD and Schema for the database can be found in the final_report.pdf file. 

### Dependencies

* Front End: React
* Back End: Express
* DataBase: MySQL
* Package Management: Node


### Installing

* The interface will be live on Oregon State classwork servers for the Fall '25 term. The interface is only accesible to Oregeon State students or staff. The source code can be downloaded and run locally to access the interface from outside Oregon State.

### Executing program

Once the frontend is built and packaged, the backend and database servers are running, the UI should be accessible via the port number specified in the files.
